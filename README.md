# TradeMind AI

AI-powered trading journal and analytics SaaS platform.

TradeMind AI helps traders:

- track trades
- analyze performance
- receive AI-generated insights
- monitor realtime market activity
- manage watchlists
- improve trading discipline

---

## Tech Stack

### Frontend
- Next.js App Router
- React
- TypeScript
- Tailwind CSS
- React Query
- Zustand

### Backend
- NestJS
- Prisma ORM
- PostgreSQL
- WebSockets
- JWT Authentication

### Infrastructure
- Nx Monorepo
- Docker
- CI/CD
- GitHub Actions

---

## Monorepo Structure

apps/
- web → Next.js frontend
- api → NestJS backend

libs/
- shared/*
- auth/*
- trades/*
- analytics/*
- ai-insights/*
- watchlist/*
- notifications/*
